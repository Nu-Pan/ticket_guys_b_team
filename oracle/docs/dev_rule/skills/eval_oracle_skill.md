
# `<tgbt-root>/oracle` の内容を評価するスキル

- 前提として `<tgbt-root>/oracle` は１００％人間の責任でメンテナンスするものであり AI による編集は禁止されている
- しかしながら、人間がメンテナンスするものなので `<tgbt-root>/oracle` に問題（論理的な矛盾など）が存在する可能性は否定出来ない
- これら「`<tgbt-root>/oracle` の問題点」の AI による評価作業はリポジトリローカルなスキルとしてパッケージングされるものとする
