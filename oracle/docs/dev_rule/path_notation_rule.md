# パス表記ルール

- `tgbt` による操作対象のリポジトリ配下のパスは `<repo-root>/...` で表記する
- `ticket_guys_b_team` リポジトリ配下のパスは `<tgbt-root>/...` で表記する
- 混同を防ぐために「リポジトリルートからの相対パス」を記述する場合は、必ず `<repo-root>`, `<tgbt-root>` を使用する
- `.codex` ディレクトリ（通常は `~/.codex`）は環境変数 `CODEX_HOME` を用いて表記する
- ユーザーホーム配下のパスは環境変数 `HOME` を使って表記する（Linux 環境を仮定して良い）
- 以上のいずれにも該当しないパス（e.g. `/usr/bin`）は絶対パスで表記する
