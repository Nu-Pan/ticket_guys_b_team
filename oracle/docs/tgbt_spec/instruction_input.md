# サブコマンドへの指示文の入力方法

## 方針
- `tgbt plan` などのサブコマンドへ指示文を入力する方法について
- 「標準入力から流し込む」と「エディタが起動して手動入力」の２パターンを用意する


## 標準入力から流し込む
- 引数として末尾に `-` が指定された場合は標準入力から指示文を取り込む
- e.g.
    ```
    cat instruction.md | tgbt plan docs --plan-id <plan id> -
    ```
- こちらはスクリプトなどの機械的な呼び出しに使う想定

## エディタが起動して手動入力

- 引数による特別な指示がない場合（デフォルト挙動）はエディタを立ち上げてユーザーに入力を促す
- 環境変数 `$EDITOR` で指定がある場合はそれを使う
- `$EDITOR` が空の場合 code --> vim --> vi の順でエディタ起動を試す
