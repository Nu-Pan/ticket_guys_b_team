## 「tgbt の開発」と「tgbt を使った開発」の混線

`ticket_guys_b_team` の開発は、ユーザーが Codex CLI を直接使って行う前提。
つまり、 tgbt の開発を tgbt で行うことはしない（禁止）。
よって、 `ticket_guys_b_team` 上の docs の読み手は人間と「`ticket_guys_b_team` を開発する Codex CLI」であり、 `tgbt` が読む前提の情報が書かれているべきではない。
この前提に違反している箇所が無いか、ドキュメントをチェックしてください。
